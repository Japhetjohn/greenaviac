# Green AVIC Website

**Redefining Hygiene, Elevating Women**

Eco-conscious feminine hygiene brand website featuring Fearless biodegradable products.

## Quick Start

### View Locally
```bash
python3 -m http.server 8000
```
Then open: http://localhost:8000

### Deploy to Hostinger

1. **Login** to Hostinger hPanel
2. **File Manager** → `public_html`
3. **Upload** all website files
4. **Enable SSL** in Security settings
5. **Done!** Visit your domain
